# Superstore Sales Performance Analysis

Falanqayntan waxay diiradda saaraysaa xogta "Superstore" si loo fahmo habka ugu wanaagsan ee kor loogu qaadi karo dakhliga ganacsiga iyo waxqabadka alaabta.

## 📂 Faylasha Mashruuca
- **`/data`**: Waxaa ku jira xogta cayriin (raw) iyo tii aan nadiifiyey (cleaned).
- **`/notebooks`**: Halkan waxaad ka heli kartaa Jupyter Notebook-ga oo sharxaya tallaabooyinka aan u maray falanqaynta.
- **`/visuals`**: Sawirrada garaafyada ee laga soo saaray xogta.
- **`/reports`**: Warbixinta PDF-ka oo dhamaystiran.

## 🚀 Maxaa laga baran karaa?
- Falanqaynta dakhliga sanadlaha ah.
- Kala saarista macaamiisha (Customer Segmentation).
- Aqoonsiga alaabta ugu dakhliga badan iyo kuwa u baahan in la horumariyo.

## 🛠 Teknoolojiyada
- Python (Pandas, Matplotlib, Seaborn)
- Excel (Data Cleaning)
